# Third-Party Notices

This plugin includes or adapts assets and ideas from the following projects:

- `muhanstudio/siyuan-excel`: independent SiYuan Excel editor structure and Luckysheet packaging approach.
- `Luckysheet`: spreadsheet editor runtime and styles.
